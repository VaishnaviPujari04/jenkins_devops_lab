### **Steps:**

### **Step 1 â†’ Create a new Pipeline Job**

### **Step 2 â†’ Add Jenkinsfile to project (recommended)**

```groovy

    agent any

    stages {
        stage('Clone Repository') {
            steps {
                git branch: 'main',

        }

        stage('Build') {

        }
    }
```
