# **4Ô∏è‚É£ Create a Dockerfile**

Inside your project folder, create a file named:

### Example Dockerfile:

```Dockerfile

```

‚úî This Dockerfile tells Docker to use NGINX and serve your website.

# **5Ô∏è‚É£ Build the Docker Image**

Run this command in the **same folder** where your Dockerfile is:

### What this command means:

* `docker build` ‚Üí builds an image
* `-t travel-agency-site` ‚Üí names the image
* `.` ‚Üí build context (current folder)
